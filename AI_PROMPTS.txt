PROMPT 1: SILVER ПРОГНОЗА (1 мач)
===========================================

Генерирай футболна прогноза за днес в JSON формат.

Изисквания:
- prediction_date: днешна дата (YYYY-MM-DD формат)
- tier: "silver"
- 1 мач с коефициент между 1.70-1.90
- Използвай реални мачове от топ лиги (Premier League, La Liga, Serie A, Bundesliga, Champions League)

JSON структура:
{
  "prediction_date": "YYYY-MM-DD",
  "tier": "silver",
  "ticket_odds": 1.80,
  "matches": [
    {
      "match_date": "YYYY-MM-DD",
      "match_name": "Отбор 1 vs Отбор 2",
      "prediction": "Тип прогноза",
      "odds": 1.80
    }
  ]
}

Валидни типове прогнози:
- "1 (Home Win)", "X (Draw)", "2 (Away Win)"
- "Over 2.5", "Under 2.5"
- "BTTS Yes", "BTTS No"
- "1X", "X2", "12"

Генерирай прогнозата сега.

===========================================
ДОПЪЛНИТЕЛНИ ИНСТРУКЦИИ ЗА AI
===========================================

Когато генерираш прогнози, моля:

1. Използвай РЕАЛНИ мачове от официални календари
2. Проверявай датите - мачовете трябва да са в правилния ден
3. Коефициентите трябва да са реалистични
4. Спазвай точно JSON формата
5. ticket_odds трябва да е произведение на всички odds
6. Всички дати в формат YYYY-MM-DD
7. tier винаги с малки букви: silver, gold, platinum
8. match_name във формат: "Отбор 1 vs Отбор 2"

===========================================
ПРИМЕРНИ ТИПОВЕ ПРОГНОЗИ
===========================================

Основни:
- 1 (Home Win)
- X (Draw)
- 2 (Away Win)

Double Chance:
- 1X
- X2
- 12

Голове:
- Over 1.5
- Over 2.5
- Over 3.5
- Under 2.5
- Under 3.5

BTTS:
- BTTS Yes
- BTTS No

Комбинирани:
- Home Win & Over 2.5
- Away Win & Over 2.5
- Home Win & BTTS
- Clean Sheet Home
- Clean Sheet Away

===========================================
BULK IMPORT WORKFLOW
===========================================

1. Избери bulk import prompt (11-20)
2. Copy/Paste в AI (ChatGPT/Claude)
3. AI генерира JSON масив с всички прогнози
4. Copy JSON-а
5. Отвори Admin → AI Import
6. Paste JSON
7. Parse → Preview
8. Кликни "Save All X" бутон
9. Готово! Всички прогнози са запазени автоматично! 🎉

===========================================
КРАЙ НА PROMPTS
===========================================

Копирай желания prompt, замени променливите (ако има) и paste в AI tool.
Резултатът ще е готов за импорт в Admin панела!

За bulk import (множество прогнози наведнъж):
- Използвай PROMPT 11-20
- Кликни "Save All" вместо "Load to Form"
- Всички прогнози се запазват автоматично!

Виж BULK_IMPORT_GUIDE.md за детайлно ръководство за bulk import.
